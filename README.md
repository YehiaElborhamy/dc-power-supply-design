# DC Power Supply Design

## Overview
A regulated DC power supply circuit designed to convert AC mains voltage into a stable DC output suitable for electronic applications.

## Features
- AC to DC conversion
- Full-wave bridge rectification
- Voltage filtering using capacitors
- Voltage regulation
- Hardware implementation and testing
- Oscilloscope and output verification

## Components Used
- Transformer
- 1N4001 Diodes
- Electrolytic Capacitors
- Voltage Regulator IC
- Fuse
- Breadboard
- Electronic Components

## Tools & Technologies

- Hardware Testing
- Electronics
- Power Electronics Basics
